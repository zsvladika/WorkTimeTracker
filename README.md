# Time Tracking System
# Zsolt Vladika 2025

This is a simple time tracking system written in Python where developers can log what they worked on daily by time.

## Features
- Logging work activities with timestamps
- Daily summaries of tracked time
- Buttons to add or subtract predefined times (+10m, +30m, +1h, -10m, etc.)
- Data saved and loaded in JSON format by day
- Simple user interface with date picker and time tracking

## Usage
1. Start the program
2. Select the date for which you want to log your work
3. Enter the task and start the timer or manually add time
4. Data is saved daily in JSON files

## Installation and Dependencies
- Python 3.x
- External libraries used:
  - `tkinter` (standard GUI toolkit)
  - `ttkbootstrap` (for enhanced UI styles)
  - `datetime` (standard for date/time handling)
  - `os` and `json` (standard libraries for file and data handling)
